# Завдання 3.5: Програма для копіювання файлів

### Функціонал
Програма є спрощеним аналогом команди `cp`. Вона виконує копіювання вмісту одного файлу в інший з попередньою перевіркою прав доступу.

### Особливості реалізації
- **Перевірка аргументів**: Програма контролює наявність обох шляхів (джерела та призначення).
- **Контроль прав**: Використано функцію `access()` для перевірки можливості читання джерела перед початком роботи.
- **Обробка сигналів**: Реалізовано перехоплення сигналу `SIGXFSZ`, що запобігає некоректному завершенню при досягненні системних лімітів на розмір файлу.
