# Завдання 3.3: Симуляція кубика з обмеженням розміру файлу

### Опис
Програма імітує кидання кубика та записує результати у файл. Для контролю ресурсів використано системний виклик `setrlimit` з параметром `RLIMIT_FSIZE`.

### Механізм обробки
При спробі записати дані, що перевищують ліміт (1024 байти), ядро Linux надсилає процесу сигнал **`SIGXFSZ`**. Програма перехоплює цей сигнал за допомогою функції-обробника, виводить повідомлення про помилку та коректно завершує роботу.
