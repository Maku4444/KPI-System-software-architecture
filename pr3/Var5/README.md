# Варіант 5: Дослідження ліміту CPU через ulimit -t

### Завдання
Встановити ліміт на використання процесорного часу за допомогою команди `ulimit -t` та запустити нескінченний цикл.

### Хід роботи
1. Написано програму на C з порожнім нескінченним циклом `while(1)`.
2. У терміналі встановлено м'який ліміт процесорного часу: `ulimit -t 2` (2 секунди).
3. При виконанні програми ядро Linux припинило її роботу рівно через 2 секунди використання ресурсів CPU.

### Результат
Програма була примусово зупинена системою. Це демонструє можливість адміністративного обмеження часу виконання "важких" або зациклених процесів на рівні командної оболонки.
