# Завдання 3.6: Дослідження ліміту стека

### Мета
Продемонструвати роботу обмеження `RLIMIT_STACK` та наслідки переповнення стека.

### Опис реалізації
- Програма використовує функцію `getrlimit` для відображення системних налаштувань стека перед початком роботи.
- Реалізовано функцію `recursive_function`, яка на кожному кроці виділяє 100 КБ пам'яті у стеку.

### Результат виконання
Програма завершується помилкою `Segmentation fault`. Це свідчить про те, що процес спробував звернутися до адреси пам'яті поза межами дозволеного стека.
