# Завдання 3.4: Симуляція лотереї з обмеженням часу ЦП

### Опис
Програма імітує вибір лотерейних чисел (7 із 49 та 6 із 36). Для контролю використання ресурсів встановлено обмеження на процесорний час (`RLIMIT_CPU`) за допомогою `setrlimit`.

### Механізм роботи
- **Ліміт**: Програмі дозволено використовувати 1 секунду процесорного часу.
- **Сигнал**: При досягненні ліміту ядро Linux надсилає сигнал **`SIGXCPU`**.
- **Обробка**: Програма перехоплює сигнал, повідомляє користувача про завершення ресурсу та припиняє обчислення.
