# Завдання 4.1: Максимальний обсяг malloc

### Опис експерименту
Запитано виділення пам'яті обсягом `(size_t)-1`, що відповідає $18446744073709551615$ байт.

### Результат
Програма вивела помилку `Cannot allocate memory`. Це підтверджує, що операційна система не може задовольнити запит, який перевищує фізичні та віртуальні ліміти.

### Висновок
Теоретичний ліміт у 16 ексабайт недосяжний, оскільки 64-бітна архітектура x86_64 ділить адресний простір: 8 ексабайт для користувача (User Space) та 8 для ядра (Kernel Space).
