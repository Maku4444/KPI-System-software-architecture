# Завдання 4.7: Безпечне виділення масивів

### Опис
Використано функцію `reallocarray(3)` для перерозподілу пам'яті під масив структур.

### Перевага
На відміну від `realloc(ptr, n * size)`, ця функція автоматично перевіряє множення `n * size` на цілочисельне переповнення.
