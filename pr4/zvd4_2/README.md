# Завдання 4.2: Цілочисельне переповнення в malloc

### Опис
Виконано множення двох великих значень типу `int`.

### Результат
Через переповнення (Integer Overflow) результат множення став рівним `0`. Функція `malloc(0)` успішно виділила пам'ять, хоча логічно очікувався запит на гіганти пам'яті.

### Висновок
Використання типу `int` для розрахунку розміру пам'яті є небезпечним. Завжди слід використовувати `size_t` та перевіряти добуток на переповнення.
