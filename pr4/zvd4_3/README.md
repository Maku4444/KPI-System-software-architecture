# Завдання 4.3: Дослідження malloc(0)

### Результати ltrace
Запуск програми через `ltrace` показав наступні виклики:
- `malloc(0)` повертає валідну адресу в купі (наприклад, `0x64f8ebde76b0`).
- `free()` успішно приймає цей вказівник.

### Висновок
У Linux бібліотека `glibc` повертає мінімальний валідний блок пам'яті навіть при нульовому запиті, що дозволяє унікально ідентифікувати об'єкт.
