# Завдання 2.1: Дослідження лімітів time_t

### Опис
Програма досліджує розрядність системного типу `time_t`. На 64-бітній системі Ubuntu цей тип займає 8 байт.

### Порівняння архітектур
- **32-bit**: Ліміт наступає 19 січня 2038 року.
- **64-bit**: Ліміт наступає через мільярди років.

### 2. Аналіз сегментів виконуваного файлу  

Використано утиліту `size` для аналізу структури файла:  

- **text**: Містить машинний код програми та константи (наприклад, рядки "ЛІМІТ ДЛЯ...").  
- **data**: Містить ініціалізовані глобальні змінні (якщо вони є).  
- **bss**: Містить неініціалізовані глобальні змінні (зарезервоване місце).
