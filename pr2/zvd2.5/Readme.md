# Завдання 2.5: Теорія стекових машин

### Чи можна обійтися без IP?
Теоретично — так. У такій архітектурі (стекова машина) замість лічильника команд використовується вершина стека. Наступна інструкція просто "виштовхується" зі стека.

### Переваги та недоліки
- **Плюс**: Компактний код, відсутність потреби у великій кількості регістрів.
- **Мінус**: Складність реалізації циклів та низька швидкість виконання через постійну роботу з пам'яттю замість регістрів.
