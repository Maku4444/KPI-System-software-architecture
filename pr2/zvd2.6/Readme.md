# Завдання 2.6: Реалізація стекової машини

### Опис моделі
Це програмна реалізація обчислювача, який використовує масив як стек для збереження операндів та проміжних результатів.
> <img width="818" height="101" alt="image" src="https://github.com/user-attachments/assets/9fc39a31-0adf-41cd-b69e-c2fde9d824b1" />


### Алгоритм роботи
Для операції `(5 + 3) * 2`:
1. `push 5`, `push 3`
2. `calc +` (результат 8 на стеку)
3. `push 2`
4. `calc *` (результат 16 на стеку)
