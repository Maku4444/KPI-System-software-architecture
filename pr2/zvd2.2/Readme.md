# Завдання 2.2: Аналіз сегментів  та оптимізація

### Результати вимірювань
Ми проаналізували, як змінюються сегменти `.text`, `.data` та `.bss` при зміні коду та прапорців компіляції.
> <img width="587" height="565" alt="image" src="https://github.com/user-attachments/assets/12ff24e2-5c12-430d-b6da-4caef229f811" />
### Висновки
1. **BSS** не збільшує розмір файлу на диску, бо це лише "резерв" пам'яті.
2. **DATA** збільшує файл, бо ініціалізовані дані зберігаються в бінарнику.
3. **Debug (-g)** створює більший файл через таблиці символів.
